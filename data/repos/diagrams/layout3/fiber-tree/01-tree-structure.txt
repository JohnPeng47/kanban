FIBER TREE STRUCTURE
====================
The internal shadow tree React maintains. Each node is a "Fiber" — a plain JS object.
Fibers link via .child (first child), .sibling (next sibling), .return (parent).
This is NOT a typical tree — it's a linked structure optimized for depth-first traversal.

Source: packages/react-reconciler/src/ReactFiber.js
