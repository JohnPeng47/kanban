EFFECT RING (circular linked list)
====================================
Effects (useEffect, useLayoutEffect) create Effect objects that go
into a SEPARATE circular linked list on fiber.updateQueue.lastEffect.

This is independent of the Hook linked list. The Hook's memoizedState
POINTS TO an Effect, but the Effects also form their own ring
for the commit phase to iterate.

Source: packages/react-reconciler/src/ReactFiberHooks.js (pushEffectImpl)
        packages/react-reconciler/src/ReactHookEffectTags.js
        packages/react-reconciler/src/ReactFiberCommitEffects.js


    fiber.updateQueue.lastEffect
          |
          v
    +----------+     +----------+     +----------+
    | Effect 1 |---->| Effect 2 |---->| Effect 3 |----.
    | useLayout|     | useEffect|     | useEffect|    |
    | Effect   |     |          |     |          |    |
    +----------+     +----------+     +----------+    |
         ^                                            |
         '--------------------------------------------'
                    (circular linked list)


    EFFECT OBJECT:

    {
      tag: HookFlags,           // bitmask: HasEffect | Passive | Layout | Insertion
      inst: {
        destroy: fn | undefined // cleanup function returned from previous run
      },
      create: () => cleanup,    // the effect function
      deps: [dep1, dep2],       // dependency array
      next: Effect,             // next in the ring
    }


    TAG BITS (which phase runs this effect):

    0b0001  HasEffect     actually needs to fire this render
                          (cleared when deps haven't changed)
    0b0010  Insertion     useInsertionEffect — runs during mutation phase
    0b0100  Layout        useLayoutEffect — runs sync after DOM, before paint
    0b1000  Passive       useEffect — runs async after paint


    DURING COMMIT, the ring is walked:

    let effect = lastEffect.next;  // first effect
    do {
      if ((effect.tag & flags) === flags) {
        // this effect matches the current phase
        if (unmounting) effect.inst.destroy();
        if (mounting)   effect.inst.destroy = effect.create();
      }
      effect = effect.next;
    } while (effect !== lastEffect.next);
