═══ SCENEINPUT EVENT FLOW ═══════════════════════════════════════════════════════════════════════════

  SVG mouseover/move/out/pointerdown (DOM listeners)
       │
       ▼
  ┌─ Tooltip system [TT] ────────────────────────────────────────────────────────┐
  │  Desktop: mouseover → closest [data-tt] → show tooltip at cursor            │
  │           mousemove → reposition                                             │
  │           mouseout → hide                                                    │
  │  Touch:   pointerdown → closest [data-tt] → show, tap elsewhere → hide      │
  │  Content: "<span class=tt-file>path</span>" + optional hint + action hint    │
  │  Fixed position div, opacity transition, z-index 100                         │
  └──────────────────────────────────────────────────────────────────────────────┘

  Viewport events
       │
       ├─ onSceneClick ──────────────────────────────────────────────────────────────
       │       │
       │       ▼
       │  ┌─ handleSceneClick [HC] ─────────────────────────────────────────────────┐
       │  │  1. Hide tooltip                                                        │
       │  │  2. hitTest(screenPoint) → hitId or null                                │
       │  │                                                                         │
       │  │  hitId == null:                                                          │
       │  │    Clear all selection visuals, fire onSelectionChange([])               │
       │  │                                                                         │
       │  │  hitId has interactive data:                                             │
       │  │    Shift+click → toggle in selection set                                │
       │  │    Normal click → replace selection with single item                    │
       │  │    CSS class "selected" toggled on SVG <g> elements                     │
       │  │    Fire onNavigate(interactive, domEvent) ──→ DiagramContentArea        │
       │  │                                                                         │
       │  │  hitId without interactive: clear selection                              │
       │  └─────────────────────────────────────────────────────────────────────────┘
       │
       ├─ onSelectionDrag ───────────────────────────────────────────────────────────
       │       │
       │       ▼
       │  ┌─ handleSelectionDrag(sceneRect) [SD] ──────────────────────────────────┐
       │  │  hitTestRect(sceneRect, "intersect") → candidate IDs                   │
       │  │  Filter to elements with interactive data                               │
       │  │  Live-update CSS "selected" class (add/remove as lasso moves)           │
       │  │  Build dragPreviewPaths for overlay labels                              │
       │  └────────────────────────────────────────────────────────────────────────┘
       │
       ├─ onSelectionDragEnd ────────────────────────────────────────────────────────
       │       │
       │       ▼
       │  ┌─ handleSelectionDragEnd(sceneRect) [SDE] ──────────────────────────────┐

       │  │  Fire onSelectionChange(elements)                                       │
       │  └────────────────────────────────────────────────────────────────────────┘
       │
       └─ onContextMenu ──→ forwarded to DiagramContentArea
              ** see below

       └─ Default ───────────────────→ fireCodeJump(interactive, domEvent)

═══ useDiagram HOOK ═══════════════════════════════════════════════════════════════════════════════

  useDiagram(html) [UD]
       │
       ├─ html changes → DOMParser.parseFromString(html, "text/html")
       ├─ Extract <svg> element
       ├─ Temporarily append to document.body (getBBox needs live DOM)
       ├─ new SvgScene(svg) → Scene
       ├─ Remove from document.body (Viewport will re-mount)
       └─ Cleanup: scene.destroy() on unmount or html change
